# 📚 About GoCart

GoCart is a modern **Multi-Vendor E-Commerce Platform** created primarily for **learning full-stack web development**.

Instead of building isolated tutorials, this project demonstrates how a real-world marketplace works—from authentication and product management to shopping carts, payments, and order tracking.

Whether you're a beginner or an intermediate developer, GoCart provides practical exposure to modern web technologies used in production.

# ⚙ Installation

Clone the repository

```bash
git clone https://github.com/Ashutosh7706/GoCart.git
```

Move inside the project

```bash
cd GoCart
```

Install dependencies

```bash
npm install
```

Create a `.env` file

```env
DATABASE_URL=

DIRECT_URL=

JWT_SECRET=

CLOUDINARY_CLOUD_NAME=

CLOUDINARY_API_KEY=

CLOUDINARY_API_SECRET=
```

Generate Prisma Client

```bash
npx prisma generate
```

Run migrations

```bash
npx prisma migrate dev
```

Start development server

```bash
npm run dev
```

Open

```
http://localhost:3000
```
